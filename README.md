# DISCORD SMRO-MarketWatch
A Discord Bot to check for new item listing.

### Getting started:

#### Configure
1. Create `conf/user_config.toml`

2. Add your `DISCORD_TOKEN` to `conf/user_config.toml`

#### Start up SMRO-MarketWatch
```bash
npm start
```
